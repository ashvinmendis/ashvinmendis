Smart Glove

This was a group project in MEng Mechatronics Engineering degree. I did the MCU programming and circuit design of the smart glove. The circuit was designed by using Eagle cad software and programming was done by using Arduino.

Smart glove is introduced to deaf people for communicate with people. when move their hand with wearing this glove it will convert sign languages to voice. A mobile phone is used for give a voice output.

The smart glove and the mobile phone is connected via Bluetooth. With using flex sensors and gyroscope sensors can be taken all the positions and angles of fingers and hand. With processing those data from microcontroller the final data value sends to mobile application via Bluetooth. Mobile application can be searched the specific data vale for voice. After mobile phone gives the voice output.
